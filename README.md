## 考核任务

基于ROS2与OpenCV4实现装甲板的识别 Vanguard视觉组考核题

 [Vanguard视觉组考核题-ArmorDetection.md](https://github.com/CodeAlanqian/vision-recruit-task-2023-7/blob/main/Vanguard%E8%A7%86%E8%A7%89%E7%BB%84%E8%80%83%E6%A0%B8%E9%A2%98-ArmorDetection.md)

### 问题背景

RoboMaster机器人对抗中需要对敌方机器人的装甲板进行识别与定位。

装甲板的识别算法，通过处理图像找到相机视野范围内的装甲板（相机一般安装在云台上，和枪口平行放置并指向同一个方向，类似瞄准镜），进而向下位机（STM32等用于控制的MCU, microcontroller unit,单片机）发送此装甲板的相对枪口的角度数据，电控根据此数据控制电机自动转向目标装甲板，实现装甲板的自动打击。

![img](https://img1.imgtp.com/2023/07/31/POyKj2Wv.png)

 

### **任务介绍**

设计一套识别算法，对提供的视频素材中的装甲板进行识别并框选，同时还要识别装甲板的颜色



### 具体要求

- 基于Ubuntu22.04环境，使用ROS2与OpenCV4进行开发
- 语言要求：C++
- vision task.tar里面的内容是视频素材，把文件放进虚拟机之后，需要用Linux命令进行解压，然后要用到ROS2 里面的`rosbag play`命令进行播放 
- 详细记录实现过程，用markdown写文档



### 考核时间

2023年8月1日-8月21日
